### 已实现功能

- [x] 戳一戳实现查看服务器状态
- [x] 微博热搜
- [x] 自动回复
- [x] 黑名单管理
- [x] 每日一图
- [x] 闪照拦截
- [x] 二维码生成
- [x] 翻译插件
- [x] 点歌插件
- [x] setu
- [x] 入群欢迎
- [x] 插件管理
- [x] 群管插件，例如禁言，关键词撤回，自动踢人
- [x] github自动操作以及搜索
- [x] 搞笑gif动画
- [x] 智能闲聊 ps:青云客api
